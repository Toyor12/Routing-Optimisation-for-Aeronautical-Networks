# Routing Optimisation for Aeronautical Networks

Graph-based routing optimisation for aeronautical networks using Dijkstra 
and extended Dijkstra algorithms to handle 216 simultaneous aircraft while 
maximising throughput and minimising latency.

## What this project demonstrates

- Graph theory and shortest path algorithms (Dijkstra)
- Network optimisation under real-world constraints
- Python implementation of complex algorithmic solutions
- Applied mathematics for aeronautical network routing

## Tech Stack

- Python
- NetworkX
- Jupyter Notebook
- NumPy / Matplotlib
